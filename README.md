### 1. Research-Based Lesson Design

**Fraction-Decimal Equivalence Lesson**  
*Grade 6 Mathematics | ~60-minute lesson*

A comprehensive, inquiry-based lesson using physical fraction strips to help students discover equivalent fractions and fraction-decimal connections through guided exploration.

**Key Features:**
- 5E instructional model (Engage, Explore, Explain, Elaborate, Evaluate)
- Hands-on manipulatives for concrete understanding
- Formative assessment checkpoints throughout
- Differentiated scaffolding strategies
- Metacognitive reflection activities
- Comprehensive 48-page teacher guide with materials

**Created for:** EDU T543 - Applied Cognitive and Learning Sciences, Harvard GSE

#### Integer Operations Space Game
**Grade 6 | Signed Numbers**

Three-card matching system teaching positive and negative integers using spatial metaphors.

**Pedagogical Features:**
- Multiple representations (numeral, visual, written description)
- Space elevator metaphor (up/down for positive/negative)
- Visual number line with color coding
- Active discovery through gameplay

**Design Rationale:** Spatial reasoning (vertical space) makes abstract negative numbers concrete. Matching multiple representations builds flexible mental models beyond rote memorization.

---

#### Number System Converter
**Grade 6 | Fractions and Decimals**

Interactive fraction visualizer using manipulable fraction strips with instant visual feedback.

**Pedagogical Features:**
- Visual fraction strips (halves, thirds, fourths, fifths)
- Click to compare and find equivalent fractions
- Instant decimal conversion showing relationships
- Pattern discovery through interaction

**Design Rationale:** Concrete-to-abstract progression where students physically manipulate visual models before moving to symbolic notation. Discovery-based learning reveals mathematical patterns rather than relying on memorization.
